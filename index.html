There's a new kind of coding I call "vibe coding", where you fully give in to the vibes, embrace exponentials, and forget that the code even exists.

I'm coding with Cursor + Claude Sonnet, voice dictating instructions with SuperWhisper, listening to new Daft Punk Tron soundtrack, hitting "Accept All", and not reading the code as it grows beyond understanding.

I had a beautiful app running in 30 minutes (that I've wanted for a while), on second iteration it broke, and I said "lol" and gave up. But it was fun, felt like the future. Totally plausible for throwaway weekend projects.
